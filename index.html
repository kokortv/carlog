📖 Настройка Apps Script
Шаг 1. Создайте Google Таблицу
Откройте sheets.google.com и создайте новую таблицу. Листы создадутся автоматически.

Шаг 2. Apps Script
В таблице: Расширения → Apps Script
Удалите всё и вставьте код из файла code.gs (прилагается).

Шаг 3. Разверните
Нажмите Развернуть → Новое развёртывание
• Тип: Веб-приложение
• Выполнять как: Я
• Доступ: Все
Скопируйте URL и вставьте в поле выше.

Шаг 4. Вшейте URL
Откройте index.html, найдите const HARDCODED_SCRIPT_URL и вставьте ваш URL между кавычками. Загрузите файл на хостинг.

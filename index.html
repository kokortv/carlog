Первый запуск
Подключите Google Sheets через Apps Script.
API ключ не нужен — авторизация автоматическая.
1
Создайте Google Таблицу
Откройте sheets.new, создайте новую таблицу и дайте ей название, например CarLog.
2
Откройте Apps Script
В таблице: Расширения → Apps Script
Удалите весь код в редакторе и вставьте содержимое файла Code.gs из этого архива.
3
Запустите setupSheets()
В редакторе Apps Script выберите функцию setupSheets и нажмите ▶ Выполнить. Разрешите доступ к таблице.
4
Разверните как Web App
Развернуть → Новое развертывание
Тип: Веб-приложение
Выполнять как: Я
Доступ: Все
Нажмите Развернуть, скопируйте URL.
5
Вставьте URL ниже
Вставьте URL вида https://script.google.com/macros/s/.../exec
